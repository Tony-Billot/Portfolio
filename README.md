Personal portfolio website built with HTML, CSS and JavaScript.

## About

This repository contains the source code of my personal portfolio. It showcases my background, skills, projects, and contact information through a modern and responsive interface.

The website was developed without any frontend framework to focus on clean, maintainable code and a lightweight user experience.

## Features

* Responsive design for desktop and mobile devices
* Presentation page with personal information
* Career and education timeline
* Skills and projects section with filtering
* Featured projects showcase
* Contact page with social links
* Privacy-friendly analytics using Plausible

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Running Locally

1. Clone the repository:

```bash
git clone https://github.com/Tony-Billot/Portfolio.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

Alternatively, you can use a local web server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
